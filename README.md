# Tiny-Polynomial-Calculator

## 一元多项式的表示和相关运算的实现 

### 【问题描述】

对于一元 n 次多项式𝑃𝑛
(𝑥) = 𝑎1𝑥
𝑒1 + 𝑎2𝑥
𝑒2 + ⋯ + 𝑎𝑚𝑥
𝑒𝑚（n＝e1＞e2＞…＞em≥0）用适当的数据结
构表示。并要求对其实现求导、给定 x 值后求多项式的值和四则运算。

### 【基本要求】

要求对一元 n 次多项式进行求导，给定 x 的具体值后求多项式的值，实现两个多项式的相减和相乘
运算。为了方便多项式的建立和显示，要求编写独立的函数用于建立多项式和显示多项式。

### 【测试数据】
